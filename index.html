Hero Vs Monster Game

This application is intended for the demonstration purposes for JavaScript classes that is paired with my blog article.
Please disregard the lack of a UI. Happy learning :)

Description: User can click the attack button under Hero or Monster to give the opponent and random number of attack points which will be deducted from the Life points.
Only the Hero is able to Heal. First one to zero Life will be the loser. Refresh the browser to start over.
